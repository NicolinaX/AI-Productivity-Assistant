
# AI Workplace Productivity Assistant 

## Project Overview
The AI Workplace Productivity Assistant is an AI-powered application designed to improve productivity by helping users manage workplace communication, meetings, and daily tasks more efficiently.
The platform combines multiple AI tools into one workspace, allowing users to automate repetitive activities, improve organisation, and save time. This project demonstrates the practical use of Artificial Intelligence in solving everyday workplace challenges.

## Features

### Smart Email Generator
- Generates professional emails based on user input.
- Allows users to select different writing tones such as formal, friendly, or persuasive.
- Helps rewrite, improve, shorten, and expand email content.

### Meeting Notes Summarizer
- Converts lengthy meeting notes into clear summaries.
- Extracts important discussion points.
- Identifies decisions and action items.
- Helps users quickly understand meeting outcomes.

### AI Task Planner & Scheduler
- Allows users to create and manage tasks.
- Prioritises tasks based on urgency and importance.
- Helps users organise daily and weekly schedules.
- Tracks task progress and deadlines.

### AI Chat Assistant
- Provides an interactive AI assistant for workplace support.
- Helps users plan tasks, generate content, and improve productivity.
- Provides AI-powered suggestions to support better workflow management.

## Tools Used

- **Lovable** – Used to design and develop the AI-powered application.
- **Artificial Intelligence (AI)** – Used to automate productivity tasks and generate intelligent responses.
- **Large Language Models (LLMs)** – Used for content generation and AI assistance.
- **Natural Language Processing (NLP)** – Used to understand and process user inputs.
- **GitHub** – Used for source control, project management, and documentation.
- **Web Development Technologies** – Used to create the application interface.

### Steps

1. Clone the repository:
   git clone https://github.com/NicolinaX/AI-Workplace-Productivity-Assistant.git

2.Open the project folder:
   cd AI-Workplace-Productivity-Assistant

3. Install project dependencies:
   npm install

4. Start the application
   npm run dev

5. Open the local development link provided in your browser to access the application.
   This is ready to paste as your **README.md** file on GitHub.
